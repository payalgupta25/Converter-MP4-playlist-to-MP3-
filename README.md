# 🎵 YouTube Playlist to MP3 Converter

Convert entire YouTube playlists into downloadable MP3 files — fast, simple, and free!

## 🛠️ Tech Stack

- **Frontend:** React (Vite)
- **Backend:** Express.js (Node)
- **Downloader:** [yt-dlp](https://github.com/yt-dlp/yt-dlp)
- **Deployment:** [Render]()
